## Description

Inventory Rest api version with Spring boot

## Required dependencies

* Java 17+ (spring boot => 3.1.5)

## Running the app

```bash
# development
$ ./gradlew bootRun
```

### Test
```bash
# Run all tests
$ ./gradlew test

# test coverage
$ ./gradlew testCodeCoverageReport
```
* you can found coverage report at build/reports/jacoco/testCodeCoverageReport
